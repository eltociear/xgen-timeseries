Numba - an overview#

Numba is a just-in-time compiler for Python that works best on code that uses NumPy arrays and functions, and loops. XGenTS includes Numba as an optional dependency and a number of functions have been included in utils.py for systems in which Numba is pre-installed. Additional functionality, XGenTS.Numba, of disabling/re-enabling numba for systems that have Numba installed has also been included.

A simple example to display the effectiveness of Numba#

import xgen as xg
import numpy as np
import timeit

from XGenTS.utils import conditional_jit, Numba
from XGenTS.stats.diagnostics import ks_summary

data = np.random.randn(1000000)

def variance(data, ddof=0):  # Method to calculate variance without using numba
    a_a, b_b = 0, 0
    for i in data:
        a_a = a_a + i
        b_b = b_b + i * i
    var = b_b / (len(data)) - ((a_a / (len(data))) ** 2)
    var = var * (len(data) / (len(data) - ddof))
    return var
